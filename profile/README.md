# FudanNLP — 复旦大学自然语言处理实验室

> 👋 Welcome to the GitHub organization of the **Natural Language Processing Lab at Fudan University**, led by [Prof. Xipeng Qiu](https://xpqiu.github.io/).
>
> 👋 欢迎来到 **复旦大学自然语言处理实验室**(由邱锡鹏教授带领)的 GitHub 主页。

Our research spans pre-trained language models, dialogue, multilingual NLP, and large language models.

研究方向涵盖预训练语言模型、对话系统、多语言 NLP 与大语言模型等。

> 🚀 For our latest work on large language models, please follow [**@OpenMOSS**](https://github.com/OpenMOSS).
>
> 🚀 最新大模型相关项目请关注 [**@OpenMOSS**](https://github.com/OpenMOSS)。

## 📦 Flagship projects / 代表项目

| Project | Description |
| --- | --- |
| [**fastNLP**](https://github.com/FudanNLP/fastNLP) | A modular and flexible NLP toolkit / 模块化、灵活的 NLP 工具包 |
| [**fastHan**](https://github.com/FudanNLP/fastHan) | All-in-one Chinese NLP toolkit / 一站式中文 NLP 工具包 |
| [**fitlog**](https://github.com/FudanNLP/fitlog) | Experiment management for deep learning / 深度学习实验管理工具 |
| [**CPT**](https://github.com/FudanNLP/CPT) | Pre-trained model for Chinese understanding and generation / 中文理解与生成预训练模型 |
| [**ElasticBERT**](https://github.com/FudanNLP/ElasticBERT) | Multi-exit pre-trained language model / 多出口预训练语言模型 |
| [**fastSum**](https://github.com/FudanNLP/fastSum) | Toolkit for neural summarization / 神经摘要工具包 |
